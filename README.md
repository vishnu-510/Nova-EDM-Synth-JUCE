# Nova EDM Synth (JUCE)

Nova is a custom EDM synthesizer plugin built using the JUCE framework.

## Features
- ADSR envelope
- Stereo widening
- Built-in effects (Reverb, Delay, Chorus)
- MIDI keyboard support
- Custom animated UI

- C++
- JUCE Framework
- VST3 / AU
